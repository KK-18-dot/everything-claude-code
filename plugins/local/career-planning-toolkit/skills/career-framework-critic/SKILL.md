# Career Framework Critic

Use this skill inside the `career-planning` repo.

If needed, read `references/review-checklist.md`.

## Workflow

1. Lock the version under review.
2. Read the framework, rubric, schema, scoring logic, and a representative report.
3. Separate framework, schema, code, and data questions.
4. Recommend the smallest change that restores coherence.
