# Session Hygiene

## Rule

Protect context quality by keeping one task per session and resetting early when stuck.

## Operational Triggers

- If the same approach fails twice, stop and reframe the task.
- If unrelated topics start mixing, split into separate sessions.
- If context contains stale assumptions, reset and restate constraints.

## Recommended Actions

1. Use `/compact` before closing a long session.
2. Save a short handoff note: goal, progress, blockers, next command.
3. Use `/clear` before starting a different task.

## Anti-Patterns

- Kitchen-sink sessions with multiple unrelated goals
- Repeating fixes without changing hypothesis
- Long free-form discussion without explicit next action
