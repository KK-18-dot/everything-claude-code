# Harness Pilot Lane

This directory is a project-local experimental lane.

Use it when normal execution is not enough and you want explicit artifacts for:

- scope
- review
- handoff
- decision

## Standard Files

1. `spec.md`
2. `sprint-01-contract.md`
3. `sprint-01-review.md`
4. `handoff.md`
5. `two-stage-decision.md`

## Rules

- keep the lane narrow
- prefer one bounded sprint at a time
- store conclusions in files, not only in chat
- do not convert this lane into the project default without a separate decision
