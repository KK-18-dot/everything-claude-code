# Evaluator Rubric

Use the smallest rubric that matches the task.

## Research / Decision Work

- evidence quality
- decision clarity
- traceability from source to conclusion
- next-step usefulness

## Code / Product Work

- functional completeness
- correctness
- test or verification strength
- operability

## Meta Checks

- did the pilot catch something the normal lane would likely miss
- did artifact handoff reduce confusion
- did the process stay bounded
- was the overhead justified by the result
